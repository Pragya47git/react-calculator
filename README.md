# React Calculator

A simple and modern calculator built with React.js. This app supports basic arithmetic operations with a clean, responsive dark-themed user interface and interactive buttons.

## Features

- Addition, subtraction, multiplication, division, and percentage  
- Clear and equals functionality  
- Responsive design for desktop and mobile  
- Smooth button animations and clean layout  
- Developer credit displayed in the footer  

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine

### Installation

Clone the repository and install dependencies:

git clone https://github.com/your-username/react-calculator.git
cd react-calculator
npm install

text

### Running the app

Start the development server:

npm start

text

Open your browser and visit `http://localhost:3000` to view the app.

## Deployment

You can deploy this project to GitHub Pages or any hosting platform supporting static React apps.
